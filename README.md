# Proyecto de Dashboard de Estadísticas de Baloncesto

Este proyecto utiliza Streamlit, Plotly y pandas para explorar y comparar estadísticas de jugadores de baloncesto de varias ligas (NBA, NCAA, WNBA).

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/basket_dashboard.git
   ```

2. Crea un entorno virtual e instala las dependencias:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   pip install -r requirements.txt
   ```

3. Ejecuta la aplicación:
   ```bash
   streamlit run app/Home.py
   ```

## Descripción

- Análisis de rendimiento de jugadores de baloncesto.
- Soporte para comparar trayectorias de jugadores de NBA, NCAA y WNBA.
